🇺🇸 English MerkepOCR is a powerful, completely free Optical Character Recognition (OCR) tool. This repository contains the portable version of the application, which runs without the need for installation.

🇺🇸 Security Warning (Windows SmartScreen)
When you download the application from GitHub, Windows may show a warning saying "Windows protected your PC". This is a standard security procedure because the application is not signed with a digital certificate.

Follow these steps to run the application:

Right-click on the Merkep-OCR.exe file and select Properties.
At the bottom of the General tab, check the "Unblock" box.
Click Apply and then OK.
Alternative Method: When the blue warning window appears, click "More info" and then click the "Run anyway" button.
✨ Key Features Portable: No installation required; keeps your registry clean. Free to Use: Access all features without any costs. Lightweight: Fast performance with minimal system resource usage.

📥 How to Download? 
Click the link https://github.com/merkepyazilim/MerkepOCRportable/releases/download/v1.0.0/MerkepOcrPortable.zip
The file will be saved to your "Downloads" folder.

🚀 Usage Instructions Right-click the downloaded .zip file and extract it to a folder of your choice. Launch the application by double-clicking MerkepOCR.exe inside the folder. No installation needed; you can carry the folder in a USB drive and use it anywhere.
