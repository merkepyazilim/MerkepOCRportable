🇺🇸 English
MerkepOCR is a powerful, completely free Optical Character Recognition (OCR) tool. This repository contains the portable version of the application, which runs without the need for installation.

## 🇺🇸 Security Warning (Windows SmartScreen)
When you download the application from GitHub, Windows may show a warning saying "Windows protected your PC". 
This is a standard security procedure because the application is not signed with a digital certificate.

**Follow these steps to run the application:**
1.  Right-click on the **Merkep-OCR.exe** file and select **Properties**.
2.  At the bottom of the **General** tab, check the **"Unblock"** box.
3.  Click **Apply** and then **OK**.
4.  **Alternative Method:** When the blue warning window appears, click **"More info"** and then click the **"Run anyway"** button.
   
✨ Key Features
Portable: No installation required; keeps your registry clean.
Free to Use: Access all features without any costs.
Lightweight: Fast performance with minimal system resource usage.

📥 How to Download?
Click the green "Code" button at the top right of this page.

🚀 Usage Instructions
Right-click the downloaded .zip file and extract it to a folder of your choice.
